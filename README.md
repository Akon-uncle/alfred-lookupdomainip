Alfred域名IP速查
==========



####使用方法

* 需要Alfred Powerpack支持,请支持正版。

下载`lookupdomainip.alfredworkflow`,安装。

![](https://luoleiorg.b0.upaiyun.com/source/images/cip1.jpg)

调起alfred，输入`cip {你要查找的URL即可}`

* 可直接输入完整URL，会自动解析出域名。

![](https://luoleiorg.b0.upaiyun.com/source/images/cip2.jpg)

回车即可复制IP地址到剪贴版。
